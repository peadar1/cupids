# Cupid's Matcher

A web application for managing college cupids matching events.

## Features
- Multi-event support
- Dynamic customizable forms
- Intelligent matching algorithm
- Collaborative matching interface
- Venue management with age restrictions
- Automated notifications

## Tech Stack
- **Backend:** Python, FastAPI, SQLAlchemy, SQLite
- **Frontend:** React, TailwindCSS

## Database Schema
- Matchers (users who create/manage events)
- Events (cupids events)
- Participants (people signing up)
- Matches (paired participants)
- Venues (pubs/locations)
- Form Questions (dynamic form configuration)
- Matching Weights (scoring configuration)
